HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element appears on the browser's title bar.
  • An element is the entire element and a tag is what appears inside the brackets.
  • Child elements are inside the body elements.
  • ul is unordered list, ol is ordered list.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A box-shadow adds shadow effects around an element's frame.
  • We can refer to a color by its hexadecimal code or hex #.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch name: creates a new branch and switches to it.
  • git checkout main: takes you the main branch.

JavaScript

  • A variable is a named container that allows us to store data in our own code.
  • Control flow is the order in which a computer executes code in a script.
  • 'Math' selects a random number
  • A function's name is followed by parentheses ()
  • The code to be executed is written inside the curly brackets {}
